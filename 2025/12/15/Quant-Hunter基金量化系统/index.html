【实战】拒绝当韭菜！我用 Python 写了个自动盯盘机器人

一句话总结：
上课没空看盘？跌了不敢加仓？
我用 Python 搞了个自动化系统，每天下午 2:45 自动帮我算账，还发微信提醒我“快跑”还是“抄底”。

为什么要做这个？

作为光电专业的学生，咱们最不缺的就是逻辑，最缺的是时间。
之前我买基金纯靠感觉，看到红了就追，看到绿了就慌。结果就是：凭运气赚的钱，凭实力亏回去了。

我就想：能不能用代码帮我克服“追涨杀跌”的人性弱点？

于是，就有了这个项目 —— **Quant-Hunter (量化猎人)**。

它能干什么？

简单说，它就是我的私人基金经理，24小时不睡觉那种。

1. 自动算账，拒绝盲操

它会自动抓取天天基金的数据，结合我设定的“多因子模型”（其实就是看趋势、看回撤、看乖离率），给每个基金打分。

  • 分数高：说明现在是黄金坑，建议买入。
  • 分数低：说明涨太多了有风险，建议止盈。
  • 熔断报警：如果跌破了 60 日均线，直接弹窗警告我“别接飞刀！”。

2. 效果演示（实拍）

这是我在本地终端运行时的样子。看着数据飞速滚动，算出最终得分的那一刻，那种掌控感真的爽。

核心逻辑（说人话版）

别被那些金融名词吓到了，其实底层逻辑就三句话：

  1. 顺势而为：站上 60 日均线才玩，跌破了就跑。（Python 里用 pandas.rolling 算均线）
  2. 不仅要猛，还要稳：赚得多不算牛，回撤小才是真牛。（用 Calmar 比率筛选）
  3. 即使是好东西，太贵也不买：涨得离谱了就要回调，这时候不能追。（用乖离率 Bias 控制）

为了防止代码太乱，我上了 Python 的类型检查。这不仅看起来专业，还能防止数据类型出错。

# 引入必要的库
from typing import Dict, Optional, Union
import pandas as pd

# 核心打分逻辑（简化版）
def calculate_score(price: float, ma60: float) -> int:
    """
    根据当前价格和均线计算得分
    """
    # 1. 如果跌破生命线，直接判死刑
    if price < ma60:
        return 0 
    
    # 2. 如果乖离率太大（涨太猛了），扣分
    bias = (price - ma60) / ma60
    if bias > 0.2:
        return 50 # 即使是牛市也要减仓
        
    # 3. 正常情况，给个满分推荐
    return 100
